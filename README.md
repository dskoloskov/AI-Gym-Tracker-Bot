# 🏋️‍♂️ AI Gym Time Tracker & Analytics Bot

## 📌 О проекте
Telegram-бот для автоматического трекинга тренировочного процесса и анализа временных рядов. Бот избавляет от необходимости вручную засекать время отдыха и сам высчитывает продуктовую метрику — «плотность тренировки» (Training Density).

## 🛠 Стек технологий
* **Backend:** Python, `pyTelegramBotAPI`
* **Database:** `SQLite3` (Проектирование реляционной БД)
* **NLP & Parsing:** Регулярные выражения (`re`) для извлечения сущностей из сырого текста
* **Data Analytics:** `pandas` (Анализ временных рядов, вычисление Timedelta)
* **Data Visualization:** `matplotlib` (Генерация дашбордов в памяти `io.BytesIO`)

## ⚙️ Ключевой функционал:
1. **Smart NLP-Дневник:** Пользователь пишет данные в свободной форме (например: "Жим лежа 100 на 8"). Алгоритм сам извлекает Упражнение, Вес и Повторения и кладет в SQL-базу.
2. **Auto Rest-Timer:** При добавлении нового подхода скрипт делает SQL-запрос к предыдущей записи, вычисляет дельту времени (Time Series Analysis) и автоматически сообщает время отдыха.
3. **Session Analytics:** По команде `/session` бот агрегирует данные за текущий день и высчитывает тренировочный объем, среднее время отдыха и индекс плотности тренировки (кг/мин).
4. **Visual Reports:** Генерация столбчатых диаграмм (отсортированных по закону Парето) для анализа недельного тоннажа по конкретным движениям.
